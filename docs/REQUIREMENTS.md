# 热点监控工具 - 需求文档

## 项目背景

作为AI编程博主，需要第一时间获取热点信息（如AI大模型更新），不能依赖人工手动搜索，而是利用工具自动发现指定的热点变化或最新热点，并及时发送通知。

## 功能需求

### 1. 关键词监控
- 用户手动输入要监控的关键词
- 当关键词相关的内容真正出现时，第一时间发送通知
- 当系统从各个信息源获取内容后，会将每条信息发送给AI进行分析，AI的审核流程：原始信息 → AI 分析 → 真实性判断 → 相关性评分 → 重要性分级 → 摘要生成
- 利用AI识别假冒/虚假内容，过滤噪音

### 2. 热点发现
- 每隔一段时间，自动搜集用户指定范围（如"AI编程"）内的热点
- 展示热点列表，支持用户查看和管理

### 3. 通知服务
- 浏览器通知（Web Push）
- 邮件通知
